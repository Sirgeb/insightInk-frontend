# InsightInk – Capture Your Day with AI

InsightInk is an **AI-powered journaling and task management application** that helps users capture, organize, and reflect on their daily activities using **voice input and artificial intelligence**.

Users can simply **record their daily activities**, and the AI will **transcribe, organize, and convert them into structured journal notes and tasks**.

---

## 📌 Project Description

InsightInk simplifies journaling and task management by allowing users to **speak about their day instead of typing it**. The app leverages AI to transform spoken input into **organized tasks and notes**, helping users stay productive and reflective.

---

## 🎯 Target Users

- **Students** managing assignments and study schedules
- **Professionals** tracking work tasks and deadlines
- **Anyone** seeking a simple AI-assisted journaling or task management system

---

## ✨ Features

### 📝 Task Management

- Add and delete **AI-generated tasks via voice**
- **Tag tasks** for easy filtering
- **AI-assisted task suggestions** and auto-generation
- **Real-time updates** with optimistic UI for immediate feedback
- **Audio recording** to capture tasks through voice input

### 👤 User Management

- Register new users
- User authentication (**login/logout**)
- Upload and update **profile picture**
- Update **personal user information**

---

## 🧠 AI Integration

InsightInk uses the **OpenAI API** to:

- Generate structured tasks from voice input
- Suggest relevant tasks
- Automatically organize spoken journal entries

---

### Frontend

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Sirgeb/insightInk-frontend.git
cd insightInk-frontend
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Development Server

```bash
npm run dev
```
